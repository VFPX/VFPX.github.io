[ReadTypeLibsAndCreateIncludeFile](https://github.com/ToniKoehler/ReadTypeLibsAndCreateIncludeFile) can create header files (.h) with all constants and enums. This is a better and faster solution than dragging and dropping all constants and enums from the Object Explorer into a .prg file.
